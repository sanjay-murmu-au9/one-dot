# PROJECT CONTEXT - BRANDING V2 & PLAYSTORE PREP

## Core Objective
Transform "One-Dot" from a basic minimalist wallpaper generator into a **Premium Stoic Sync Engine**. This phase focuses on high-fidelity aesthetics, professional branding, and preparing the app for its first production release on the Google Play Store.

## Key Themes: "Premium Stoic"
- **Obsidian (#0b0b0c)**: A deep, true-black palette inspired by high-end hardware.
- **Paper White (#fdfdfc)**: A soft, textured off-white palette for a clean, non-glare look.
- **Stoic Typography**: 
    - *JetBrains Mono*: For technical, dot-matrix vibes.
    - *Playfair Display*: For classical, mindful elegance.

## Play Store Readiness
- **Production Package**: `com.sanjayamurmu.onedot`
- **Native Experience**: Automatic device detection (OnePlus, iPhone, etc.) must feel seamless, with mockups that match the user's physical hardware.
- **Branding Assets**: Visuals must emphasize the "One-Click Sync" across multiple devices.

## Technical Goals
1.  **Aesthetic Excellence**: Every UI element must feel premium (smooth transitions, refined spacing, curated colors).
2.  **Mobile-First Optimization**: Simplify the UI on mobile (hiding manual toggles) to make it feel like a utility rather than a complex website.
3.  **Production Stability**: Finalize app identification and build configurations for the Google Play Console.
